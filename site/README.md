# CX Website

This public documentation site is built with [Docusaurus](https://docusaurus.io/).

## Installation

```bash
pnpm install
```

## Local Development

```bash
pnpm run start
```

## Build

```bash
pnpm run build
```

## Deployment

GitHub Pages deployment is handled by `.github/workflows/pages.yml` on pushes to
the `docs` branch.
