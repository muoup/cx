# C Syntax
## (Supported and To Be Supported)

### Supported

- Program := { Function }
- Function := Type Identifier '(' ')' '{' { Expression } '}' 
- Expression := KeywordExpression | IdentifierExpression;
- KeywordExpression := 'return' Identifier ';'
- IdentifierExpression := 
- Type := 'int' | 'void'
- Identifier := \[a-zA-Z_]\[a-zA-Z0-9_]*
